Sort By

Each data category has its own set of fields by which that dataset can be sorted:

  • localhost:3000/films
    sortBy=episode_id Default
    sortBy=title
    sortBy=release_date
  • localhost:3000/people
    sortBy=name Default
    sortBy=height
    sortBy=mass
  • localhost:3000/planets
    sortBy=name Default
    sortBy=diameter
    sortBy=population
  • localhost:3000/species
    sortBy=name Default
    sortBy=height
    sortBy=lifespan
    • <>localhost:3000/starships
    sortBy=name Default
    sortBy=cost
    sortBy=length
  • localhost:3000/vehicles
    sortBy=name Default
    sortBy=cost
    sortBy=length

Sort Order

By default, all data is sorted in ascending order.

The sort order can be specified explicitly by adding the query string parameter sortOrder=asc or reversed by adding sortOrder=desc.
