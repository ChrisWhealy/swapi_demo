  • /species
  • /starships
  • /vehicles

Sort By

Each data category has its own set of fields by which that dataset can be sorted:

  • localhost:3000/films
    • sortBy=title
    • sortBy=director
    • sortBy=episode
  • localhost:3000/people
    • sortBy=name
    • sortBy=homeworld
    • sortBy=height
  • localhost:3000/planets
    • sortBy=name
    • sortBy=rotation
    • sortBy=orbit
  • localhost:3000/species
    • sortBy=name
    • sortBy=language
    • sortBy=height
    • sortBy=lifespan
  • localhost:3000/starships
    • sortBy=name
    • sortBy=cost
    • sortBy=length
  • localhost:3000/vehicles
    • sortBy=name
    • sortBy=cost
    • sortBy=length
    • sortBy=manufacturer

Sort Direction

Unless stated otherwise, all data is sorted by name (or title) in ascending order.

The sort order can be specified explcitly by adding the query string parameter sortDir=asc or reversed by adding sortDir=desc.
